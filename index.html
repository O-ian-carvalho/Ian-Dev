myGains é uma aplicação web feita em python, utilizando o framework flask. A ideia da aplicação é permitr que os usuário acompanhem a evolução nos treinos, tenham acesso aos dados de musculos, exercicios, cargas repetiçoes etc. Também permitindo que o usuário compare a sua evolução com a evolução dos outros usuários.
